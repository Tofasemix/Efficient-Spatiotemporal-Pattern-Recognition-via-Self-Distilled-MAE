# Efficient Spatiotemporal Pattern Recognition via Self-Distilled MAE

## Project Title
Efficient Spatiotemporal Pattern Recognition via Self-Distilled MAE

## Baseline Results
The baseline model achieves a **0.7733 Micro-AUC** on the ShanghaiTech dataset.

## Technical Highlights
- **Latent Feature Distance Extraction**: Implemented to effectively identify deviations indicating spatiotemporal anomalies.
- **Local Min-Max Normalization**: Applied specifically per video to ensure robust and bounded anomaly scores.

## Hardware
Training for this baseline was performed on an **NVIDIA RTX 4500 Ada**.
